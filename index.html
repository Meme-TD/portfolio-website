Marcus Ling Project Portfolio

This website hosts and describes three simulation-focused projects.

1) Missile Defense Simulation Tool

Models threat detection, interception decisions, and budget tradeoffs.

  • Threat generation with speed/trajectory/decoy modeling
  • Greedy, predictive, and ML-based prioritization algorithms
  • Budget constraints for interceptor use and radar coverage

Live demo: Open Project

GitHub repo: github.com/Meme-TD/missile-defense-simulator

2) Badminton Smart Training Simulator + Analytics

Tracks shots and outputs heatmaps, probabilities, and tactical suggestions.

  • Input shot type, location, and outcome
  • Shot placement + movement heatmaps
  • Success-rate analytics and optimal-shot suggestion engine

Live demo: Open Project

GitHub repo: github.com/Meme-TD/badminton-smart-training

3) Concert Show Success Simulator

Allows users to design a show and get synthetic success predictions without external data.

  • User controls genre, theme, budget, cast popularity, marketing, style, runtime, and ticket price
  • Simulates audience reception, critical reviews, and mass appeal
  • Predicts weekly revenue using a lightweight AI-style scoring agent

Live demo: Open Project

GitHub repo: github.com/Meme-TD/concert-show-success-simulator
